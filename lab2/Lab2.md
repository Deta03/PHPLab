## Контрольные вопросы

### 1. Какие способы установки PHP существуют?
- **Ручная установка** с официального сайта (php.net) — скачивание архива, распаковка, настройка PATH
- **XAMPP** — комплексное решение (Apache + PHP + MySQL + phpMyAdmin)
- **WAMP** — аналог XAMPP для Windows
- **Laragon** — легковесная альтернатива
- **Через пакетные менеджеры** (Chocolatey, Scoop)

### 2. Как проверить, что PHP установлен и работает?
```
php -v          // Показывает версию PHP
php -m          // Показывает загруженные модули
php -i          // Полная информация о конфигурации
echo <?php phpinfo(); ?>  // Веб-интерфейс (через браузер)
```

### 3. Чем отличается оператор echo от print?
| Характеристика | echo | print |
|----------------|------|-------|
| **Тип** | Языковая конструкция | Функция |
| **Возвращаемое значение** | Нет (void) | 1 (int) |
| **Аргументы** | Несколько через запятую | Один аргумент |
| **Скорость** | Быстрее | Медленнее |
| **Скобки** | Не обязательны | Рекомендуются |

**Пример различий:**
```php
echo "A", "B", "C";     // Работает (3 аргумента)
print "A", "B";         // Ошибка (1 аргумент)
$result = print "Test"; // $result = 1
// $result = echo "Test"; // Синтаксическая ошибка
```
