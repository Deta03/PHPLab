# Ход работы

## 1. Создание массива транзакций

Был создан массив `$transactions`, содержащий информацию о банковских
транзакциях (id, date, amount, description, merchant).\
Каждая транзакция хранится как ассоциативный массив.

Пример:

``` php
$transactions = [
[
"id" => 1,
"date" => "2019-01-01",
"amount" => 100.00,
"description" => "Payment for groceries",
"merchant" => "SuperMart"
]
];
```

------------------------------------------------------------------------

## 2. Вывод данных

Для вывода транзакций использовалась HTML‑таблица и цикл `foreach`,
который проходит по каждому элементу массива и выводит его значения.

# Ответы на вопросы

### 1. Что такое массивы в PHP?

Массив --- это структура данных, которая хранит несколько значений в
одной переменной. Каждый элемент имеет ключ и значение.

### 2. Как создать массив в PHP?

Массив можно создать с помощью:

``` php
$array = [1,2,3];
```

или

``` php
$array = array(1,2,3);
```

### 3. Для чего используется цикл foreach?

`foreach` используется для перебора элементов массива и выполнения
действий для каждого элемента.
