Задание
-------

Используя функцию date(), создать таблицу с расписанием сотрудников на основе текущего дня недели.

### Условия

**John Styles**

* Понедельник, среда, пятница — 8:00–12:00

* Остальные дни — Нерабочий день

**Jane Doe**

* Вторник, четверг, суббота — 12:00–16:00

* Остальные дни — Нерабочий день

$cuday = date('N');

    $janeSchedule = "12:00 - 16:00";

    $janeSchedule = "Нерабочий день";

?>
<table border="1" cellpadding="10" style="border-collapse: collapse; text-align: left;">
    <thead>
    <tr>
        <th>№</th>
        <th>Фамилия Имя</th>
        <th>График работы</th>
    </tr>
    </thead>
    <tbody>
    <tr>
        <td>1</td>
        <td>John Styles</td>
        <td><?php echo $johnSchedule; ?></td>
    </tr>
    <tr>
        <td>2</td>

    </tr>

</table>

2\. Циклы

Исходный код

------------

```php
<?php

$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
   $a += 10;
   $b += 5;
}

echo "End of the loop: a = $a, b = $b";
```

2.1 Цикл for с выводом промежуточных значений
---------------------------------------------

```php
<?php

$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
}

echo "<br>End of the loop: a = $a, b = $b";
```

2.2 Переписывание через while
-----------------------------

```php
<?php

$a = 0;
$b = 0;
$i = 0; // Инициализируем счетчик

echo "<h3>Цикл WHILE</h3>";

while ($i <= 5) {
    $a += 10;
    $b += 5;

    echo "Шаг $i: a = $a, b = $b <br>";

    $i++; // Увеличиваем счетчик, иначе цикл будет бесконечным!
}

echo "<b>Итог цикла while: a = $a, b = $b</b><hr>";

```

2.3 Переписывание через do-while
--------------------------------

```php
<?php
$a = 0;
$b = 0;
$i = 0;

echo "<h3>Цикл DO-WHILE</h3>";

do {
    $a += 10;
    $b += 5;

    echo "Шаг $i: a = $a, b = $b <br>";

    $i++;
} while ($i <= 5);

echo "<b>Итог цикла do-while: a = $a, b = $b</b>";
?>
```

3\. Контрольные вопросы

3.1 В чем разница между циклами for, while и do-while?
------------------------------------------------------

### for

* Используется, когда заранее известно количество итераций.

* Содержит инициализацию, условие и изменение счетчика в одной строке.

* Удобен для работы со счетчиком.

### while

* Используется, когда количество повторений заранее неизвестно.

* Сначала проверяется условие.

* Может не выполниться ни разу.

### do-while

* Сначала выполняется тело цикла.

* Затем проверяется условие.

* Выполняется минимум один раз.

3.2 Как работает тернарный оператор ? : в PHP?
----------------------------------------------

Тернарный оператор — это сокращенная форма условного оператора if-else.

### Синтаксис

`условие ? значение_если_true : значение_если_false;`

Если условие истинно — возвращается первое значение, иначе второе.

3.3 Что произойдет, если в do-while поставить условие, которое изначально ложно?
--------------------------------------------------------------------------------

Тело цикла выполнится **один раз**, так как:

1. Сначала выполняется блок do

2. Затем проверяется условие в while

3. Если условие ложно — цикл завершается
